## Features
- Menu system
- Custom number range 
- Modular OOP functions
	- FizzBuzzRule
	- FizzBuzzEngine
	- Range
	- Menu
- Support to add new divisors


## How to run
1. Open the project in any IDE that is in C++
2. Build/compile all the .cpp files
3. Run the resulting executable

## Usage
1. Run program
2. Choose menu options:
- Start FizzBuzz
- Set custom range
- Quit
